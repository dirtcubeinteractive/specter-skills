# Next Steps

## Where to go from here?

With the SDK now installed, configured, and initialized, you're all set to explore the extensive features the SpecterSDK offers.&#x20;

1. **Read Code Comments**: The code in the SDK is well commented and will give you a good understanding of specific functionality. Check out the [README](https://github.com/dirtcubeinteractive/SpecterSDK/blob/master/README.md) file to learn how to approach reading code comments.
2. **Read Specter Manual**: Go through and understand the Specter dashboard from the [Specter User Manual](https://dirtcube-interactive.gitbook.io/specter-user-manual/) if you haven't already done so.
3. **Read Specter API Docs**: We recommend going through our [REST API documentation](https://doc.specterapp.xyz) to understand how the SDK communicates with Specter backend services.

## Conclusion

We're excited to have you onboard and using the SpecterSDK for Unity! Our documentation has been designed to provide you with a solid foundation to get started, offering comprehensive guides on setup, configuration, and key features of the SDK.

As we continue to enhance the Specter platform, we look forward to introducing feature-specific guides and recommended strategies within this documentation. These forthcoming updates aim to further empower your development process, providing detailed insights into leveraging the SpecterSDK’s full potential.

Keep an eye out for these enhancements, designed to enrich your experience and support your success with Specter.

Happy coding!
