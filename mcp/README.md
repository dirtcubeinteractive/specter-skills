# Specter MCP server

The MCP server is part of the **`specter-skills`** package, not a standalone package. See the
[root README](../README.md#mcp-server) for configuration and tools.

Run it via: `npx -y -p specter-skills specter-mcp` (or `specter-mcp` if the package is installed).

Source: `mcp/src/index.js` (entry), `mcp/src/specter-client.js` (HTTP client).
