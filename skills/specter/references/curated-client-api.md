### Get Bundles

`POST /v2/client/app/get-bundles`

Retrieves in-game bundles that can include multiple items, currency packs, or gear sets. Useful for combining virtual goods into discounted or special-offer packages for players.

**Required headers:** `api-key`

| Param | Type | Required | Allowed values | Description |
|---|---|---|---|---|
| `bundleIds` | array<string> | — |  | Array of specific bundle IDs to look up (e.g., 'starter_pack', 'holiday_bundle'). |
| `isLocked` | boolean | — |  | Filter by locked (true) or unlocked (false) status. A locked bundle may require special conditions, like level requirements. |
| `offset` | number | — |  | Number of bundles to skip (for paginated UIs). |
| `limit` | number | — |  | Maximum number of bundles to return in a single response. |
| `search` | string | — |  | Search keyword to filter bundles by name |
| `includeTags` | array<string> | — |  | Array of tags to narrow down or categorize the bundles. |
| `attributes` | array<enum> | — | `properties` \| `prices` \| `contents` \| `unlockConditions` \| `meta` \| `tags` | Array of attributes to include in the response |
