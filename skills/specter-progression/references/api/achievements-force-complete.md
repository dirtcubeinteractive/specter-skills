# Client API (v2): `achievements/force-complete`

**Endpoint:** `POST /v2/client/achievements/force-complete`

**Tag:** Achievements

**Summary:** Force complete a task/achievement

**Auth:** Project **api-key** (`api-key` header) **+ player access token** (`Authorization: Bearer <playerAccessToken>`). Player-scoped — call after logging the player in.
