`POST /v2/client/account/change-password`

Allows the user to change their password by providing the current and new passwords.

**Required headers:** `api-key`, `Authorization`

| Param | Type | Required | Allowed values | Description |
|---|---|---|---|---|
| `currentPassword` | string | ✅ |  | The user's current password. |
| `newPassword` | string | ✅ |  | The new password the user wants to set. |


### Link Account

`POST /v2/client/account/link`

Links a new account to a user's profile, allowing the user to log in through various providers.

**Required headers:** `api-key`, `Authorization`

| Param | Type | Required | Allowed values | Description |

| `type` | enum | ✅ | `google` \| `facebook` \| `customId` | Specifies the type of account to link, such as a custom ID, Google, or Facebook account. |
| `id` | string | ✅ |  | The unique identifier of the account to be linked, specific to the account type |
